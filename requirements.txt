# AI DOCUMENT PROCESSING INSTALLATION INSTRUCTIONS:
# Uses only Python libraries - no third-party API keys required
#
# 1. Install Python dependencies:
#    pip install pytesseract PyPDF2 python-docx pyresparser spacy nltk
#
# 2. Install Tesseract OCR (for better image OCR):
#
#    WINDOWS:
#    - Download from: https://github.com/UB-Mannheim/tesseract/wiki
#    - Install the executable
#    - Add Tesseract to your PATH environment variable
#
#    LINUX (Ubuntu/Debian):
#    sudo apt-get update
#    sudo apt-get install tesseract-ocr
#
#    macOS:
#    brew install tesseract
#
# 3. Install spaCy and NLTK models (for pyresparser):
#    python -m spacy download en_core_web_sm
#    python -m nltk.downloader words
#
# 4. Test installation:
#    python test_simple_ocr.py
#
# 5. Text Extraction Methods (in order of preference):
#    - pyresparser (for resumes - most accurate)
#    - Local Tesseract OCR (for images)
#    - PyPDF2 PDF text extraction (for PDFs)
#    - python-docx text extraction (for DOCX files)
#    - Basic text file reading (for TXT files)


# FACE RECOGNITION INSTALLATION INSTRUCTIONS:
# 
# WINDOWS USERS - Choose ONE of these methods:
#
# METHOD 1 (Recommended - Pre-compiled wheels):
# 1. pip install cmake
# 2. pip install dlib --find-links https://github.com/sachadee/Dlib/releases
# 3. pip install face_recognition
#
# METHOD 2 (Alternative - Conda):
# 1. conda install -c conda-forge dlib
# 2. pip install face_recognition
#
# METHOD 3 (Manual wheel download):
# 1. Download dlib wheel from: https://github.com/sachadee/Dlib/releases
# 2. pip install dlib-19.24.2-cp311-cp311-win_amd64.whl
# 3. pip install face_recognition
#
# LINUX/MAC USERS:
# 1. pip install cmake
# 2. pip install dlib
# 3. pip install face_recognition
#
# TROUBLESHOOTING:
# - If dlib fails, try: pip install --upgrade pip setuptools wheel
